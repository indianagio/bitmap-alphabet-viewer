# Bitmap Alphabet Viewer

Viewer HTML locale per mostrare l'alfabeto completo (`A-Z` e `a-z`) in celle bitmap da **12x16 pixel**, con anteprima scalata **10x**.

## Avvio locale

Apri `index.html` direttamente nel browser.

## Funzioni

- Caricamento font `.ttf`, `.otf`, `.woff`, `.woff2`
- Griglia con lettere maiuscole e minuscole
- Cella logica fissa 12x16
- Ingrandimento 10x
- Regolazione font size, offset X/Y e soglia alpha
